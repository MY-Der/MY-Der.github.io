# 🐾 少芳的四小只 · 像素宠物乐园

一个温馨的虚拟宠物养成游戏，包含四只性格迥异的像素宠物，支持互动、事件触发和亲密度里程碑。

## ✨ 功能特点

- **四只宠物**：小胡汪（狗）、灯神喵（猫1）、璇宝咪（猫2）、可可酱（狐狸），每只有独特语录和交互偏好。
- **四项属性**：饱腹度、清洁度、心情、亲密度，会随时间自然衰减。
- **互动操作**：喂食、洗澡、玩耍、抚摸（单击宠物），可切换「全体模式」一次照顾所有宠物。
- **随机事件**：双宠组合会触发特殊剧情（如舔毛、追逐蝴蝶），影响属性并收录至相册。
- **亲密度祝福**：
  - 全员亲密度 ≥60 → 蛋糕庆生动画 + 生日歌（可手动播放）。
  - 全员属性全满 → 拍立得大合照动画 + 宠物告白。
- **事件相册**：记录所有已触发的剧情事件，可随时回顾。
- **离线衰减**：关闭页面后，宠物属性会随时间缓慢下降，模拟真实照顾感。
- **本地存档**：所有进度（属性、解锁事件、相册）自动保存于浏览器。

## 🎮 如何游玩
直接打开 `index.html` 文件（现代浏览器均可），或在当前页面网址末端输入“/zsf”

👉 点击进入「少芳的四小只」[https://my-der.github.io/zsf/](https://my-der.github.io/zsf/)

1. 单击任意宠物卡片可选中它，之后的操作（喂食等）只对该宠物生效。
2. 勾选「全体互动」后，操作将同时影响所有四只宠物。
3. 通过下方按钮进行喂食、洗澡、玩耍，宠物头顶会冒出对话气泡。
4. 查看下方日志区了解属性变化和事件提示。
5. 当亲密度达标时，底部会出现「生日蛋糕」或「回顾合照」按钮，点击触发特殊动画。
6. 点击「事件相册」可查看已解锁的剧情插画。

## 🛠️ 操作提示

| 操作 | 说明 |
|------|------|
| 单击宠物 | 抚摸该宠物，增加少量心情和亲密度 |
| 喂食 | 增加饱腹度 +30，心情少量增加 |
| 洗澡 | 增加清洁度 +40，心情少量增加 |
| 玩耍 | 增加心情 +15，减少少量饱腹和清洁 |
| 全体模式 | 切换后所有操作对四只宠物同时生效 |
| 重置按钮 | 清除所有存档并刷新页面 |
| 事件相册 | 查看已触发的剧情事件 |
| 蛋糕按钮 | 重新播放庆生动画（需亲密度全≥60） |
| 合照按钮 | 重新播放拍立得动画（需亲密度全满） |

## ❓ 常见问题

**Q：为什么宠物的属性不变了？**  
A：检查是否已触发「全满祝福」，全满状态下继续喂食等操作不会再有属性提升（但会显示提示）。

**Q：生日蛋糕或合照按钮为什么不出现？**  
A：蛋糕需要所有宠物的**亲密度≥60**，合照需要所有宠物的**亲密度全为100**。多与宠物互动即可。

**Q：如何触发双宠事件？**  
A：系统每隔约 10 分钟会自动尝试触发一个尚未解锁的双宠事件。多与宠物互动可增加亲密度，间接帮助解锁。

**Q：离线后属性下降太多怎么办？**  
A：这是设计的一部分，鼓励定期登录照顾。重新喂食或洗澡即可恢复。

**Q：如何手动备份存档？**  
A：浏览器开发者工具 → 应用程序 → LocalStorage，复制 `shaofang_pet_data` 和 `shaofang_event_album` 的值。
